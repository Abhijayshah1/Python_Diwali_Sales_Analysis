# Python Diwali Sales Analysis

A Python data analysis project focusing on Diwali sales data to improve customer experience and boost sales. This project involves data cleaning, exploratory data analysis (EDA), and data visualization to uncover actionable business insights.

## 🎯 Objective
To analyze customer purchasing behavior during the Diwali festival and identify key demographics driving sales. This allows businesses to optimize their marketing strategies, target specific consumer groups, and improve inventory management.

## 🛠️ Tools & Technologies Used
- **Python** (Programming Language)
- **Pandas** (Data Manipulation & Cleaning)
- **NumPy** (Numerical Computations)
- **Matplotlib** (Data Visualization)
- **Seaborn** (Advanced Statistical Visualizations)
- **Jupyter Notebook** (IDE)

## 🔍 Key Insights & Conclusion
Based on the exploratory data analysis, the key target demographic driving the most sales was identified:

> *Married women in the age group of 26-35 years from Uttar Pradesh, Maharashtra, and Karnataka, working in the IT, Healthcare, and Aviation sectors, are the most likely to buy products from the Food, Clothing, and Electronics categories.*

## 📝 Project Details
- **Data Cleaning & Preprocessing:** 
  - Imported a dataset containing over 11,000 sales records.
  - Handled missing values (e.g., dropped irrelevant/blank columns like `Status`, `unnamed1`).
  - Standardized data types to prepare for accurate analysis.
- **Exploratory Data Analysis (EDA):** 
  - Analyzed purchasing power and frequency by demographics such as Gender, Age Group, Marital Status, State, and Occupation.
  - Identified the highest-performing consumer segments and product categories.
- **Visualizations:**
  - Designed detailed bar plots and frequency graphs using Seaborn and Matplotlib to present the insights clearly.
